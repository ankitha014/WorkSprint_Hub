# Project Flow 🚀

A collaborative project management system to manage tasks, teams, and workflows efficiently.

---

## 📌 Features

* 🗂️ Project and workspace management
* ✅ Task creation and tracking
* 📊 Analytics dashboard
* 🔔 Notifications system
* ⚙️ User settings and customization

---

## 🛠️ Tech Stack

* Frontend: Vite + React
* Backend: Supabase
* Database: PostgreSQL
* Styling: CSS / Tailwind (if used)

---

## 📁 Project Structure

```
collab-comet/
│── src/
│── public/
│── .env.example
│── package.json
│── README.md
```

---

## ⚙️ Environment Variables

Create a `.env` file based on `.env.example`:

```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone <your-repo-url>
cd collab-comet
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
